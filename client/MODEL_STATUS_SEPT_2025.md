# AI Model Status - September 2025

Complete reference of available AI models as of September 2, 2025, organized by provider and tier.

## Tiering System

**PRIMARY TIER**: Latest models from 2025, shown by default in UI
**SECONDARY TIER**: Older models (2024 and earlier), shown under "Show all models"
**DEPRECATED**: Models being phased out or no longer available

## Key Changes Since January 2025

### Major Releases
- **Grok-4** with real-time search integration
- **Llama 4 series** with mixture-of-experts architecture
- **Claude 4 series** replacing all Claude 3.x models
- **Kimi K2** with improved coding performance
- **DeepSeek V3.1** hybrid reasoning/chat model

### Deprecations
- **All Claude 3.x models** retiring October 2025
- **Gemini 1.5 series** restricted for new projects
- **GPT-5 series** not available via API (ChatGPT only)

### Cost Optimizations
- Llama 4 models offering better performance at lower costs
- Gemini Flash-Lite variants for ultra-efficient processing
- DeepSeek models maintaining very competitive pricing
